Week 14 : Computation Distribution

Python server : Cloud computing

Connect clients to a server (TCP) and send a matrix and a colomn of the 2nd matrix. Each client computes the given data and sends back the results.
The server recieves the results and puts them together : cloud computing distribution.
The implemented server is easiy scalable, done using OOP principles, can handle clients (ping, kick, ban, send / recieve message).
The implementation is easily scalable for computing distribution in the meaning that all you have to is send a type (to identofy the request in the client side), and an array of messages (all messages to be sent and computed by clients. The server will automtically manage everything and send back an array of the responses of the clients.
To put in a nutshell, the client must only create the messages to send, wait, process recieved messages.
